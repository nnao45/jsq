🚀 JSQ Web Playground - ローカルサーバー起動方法

QuickJS WASMを使用するため、file://プロトコルでは動作しません。
以下のいずれかの方法でローカルサーバーを起動してください。

方法1: Python (推奨)

# Python 3
python -m http.server 8000

# Python 2 (古い環境の場合)
python -m SimpleHTTPServer 8000

起動後: http://localhost:8000 にアクセス

方法2: Node.js

# npx を使用（インストール不要）
npx serve .

# または http-server を使用
npx http-server -p 8000

# グローバルインストール済みの場合
serve .
# または
http-server -p 8000

方法3: VS Code拡張機能

「Live Server」拡張機能をインストール後、index.htmlを右クリック → 「Open with Live Server」

方法4: その他の方法

# Ruby
ruby -run -e httpd . -p 8000

# PHP
php -S localhost:8000

# Deno
deno run --allow-net --allow-read https://deno.land/std/http/file_server.ts

サーバーが起動したら、http://localhost:8000 でJSQ Web Playgroundにアクセスできます！
